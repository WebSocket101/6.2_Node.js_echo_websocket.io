# WebSocket-Node Echo-Server
Node.js Socket-io Echo-Server

# Dependencies
* [Node.js]: http://nodejs.org/
* [socket.io]: https://github.com/theturtle32/WebSocket-Node

# Deployment
Before first time deployment please execute:
```bash
npm install
```
Start the application by:
```bash
npm start
```

or

```bash
node echoServer.js
```
