# WebSocket-io Echo-Server
Node.js Websocket-io Echo-Server

# Dependencies
* [Node.js]: http://nodejs.org/
* [socket.io]: https://github.com/LearnBoost/websocket.io

# Deployment
Before first time deployment please execute:
```bash
npm install
```
Start the application by:
```bash
npm start
```

or

```bash
node echoServer.js
```
